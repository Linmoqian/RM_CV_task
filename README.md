# RM_CV_task

RM 视觉组面试任务仓库，包含两个方向：前后端开发、CV

## 快速开始

```bash
git clone https://github.com/Linmoqian/RM_CV_task
cd RM_CV_task
conda env create -f environment.yml
conda activate rm-cv
```

## 目录结构

```
RM_CV_task/
├── UI_design/   # 前后端开发方向的设计原则和设计稿
└── CV_task/     # 视觉组考核题
```
