# 神经网络剪枝的学习

剪枝，剪的是不重要的通道，是降本增效的一种工程方法。

降低参数量、运行内存和计算量

这篇论文提出了一种基于 BatchNorm 缩放因子 γ 的通道剪枝方法。作者通过在训练阶段对 γ 加入 L1 稀疏正则，使不重要通道的 γ 接近 0，再根据 γ 的大小进行全局阈值剪枝，并对得到的紧凑网络进行微调。与非结构化剪枝相比，该方法得到的是更窄的致密网络，因此不依赖专用稀疏硬件或软件即可获得模型压缩和推理加速。实验表明，该方法在 VGG、ResNet 和 DenseNet 上均有效，在 CIFAR、SVHN 和 ImageNet 上都能显著降低参数量和 FLOPs，并在适度剪枝时保持甚至提升精度。

目标：压缩 CNN，同时保留精度。
方法：对 BN 的 γ 做 L1 稀疏约束。
流程：稀疏训练 → 按 γ 剪通道 → 微调 → 可多轮重复。
结果：在多个数据集和结构上，显著减少参数和 FLOPs，且通常不损失精度。
